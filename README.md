- [Docs (release)](https://docs.rs/substrait)

## Build & Test

Clone this repository.

```shell
git clone --recursive git@github.com:substrait-io/substrait-rs.git
cd substrait-rs
```

Build and test with Cargo.

```shell
cargo test
```
